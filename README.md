# springMvcInternationalisation
This example presents the basic concept of using in Spring MVC internationalisation.

The technologies used are :
 - Spring 4.3.18
 - JDK 1.8
 - Maven 3.3.9

You can convert this example to an Eclipse IDE project by going to folder where is the pom.xml is, and use the command :

mvn eclipse:eclipse

Inspired from "Beginning Spring" Mert Caliskan, Kenan Sevindik, Rod Johnson (Foreword by), Jürgen Höller (Foreword by).
